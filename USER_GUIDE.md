# SlideTimer — User Guide

## How to Launch the App

Double-click **SlideTimer.app** on your Desktop (or wherever you copied it).

The control window opens immediately. The floating overlay is hidden until you click **Show Overlay**.

---

## Quick Start — 5-Minute Countdown

1. Launch **SlideTimer.app**
2. In the **Duration** field, type `05` : `00` and press Return (or just click a preset — the **5m** button works too)
3. Click **Show Overlay** — a small floating window appears on screen
4. Drag the overlay to your secondary monitor / wherever you want it visible
5. Click **Start** (or press **Space**)
6. Timer counts down. At **30 s** it turns orange. At **10 s** it turns red. At **0:00** it flashes and plays a sound.
7. Click **Reset** (or press **Cmd+R**) to return to 5:00 and start again

## All Controls Explained

### Mode
| Option | Behaviour |
|--------|-----------|
| **Countdown** | Counts down from the duration you set. Stops and flashes at 0:00. |
| **Stopwatch** | Counts up from 0:00 indefinitely. |

### Duration
Type minutes and seconds directly into the two fields, then press **Return** to apply. Disabled while the timer is running.

### Presets
One-tap shortcuts: **1 m · 2 m · 3 m · 5 m · 10 m · 15 m · 20 m · 25 m**. Clicking a preset resets the timer to that duration.

### Start / Pause / Restart
- **Start** — begins the countdown (green button)
- **Pause** — freezes the timer exactly where it stopped (amber button, shown while running)
- **Restart** — shown after the timer finishes; resets and immediately starts a fresh run

### Reset
Returns the timer to the set duration without starting it. Also available via **Cmd+R** or **Escape**.

### Appearance

| Control | What it does |
|---------|-------------|
| **Timer Color** — swatches + picker | Sets the normal timer text color in the overlay and control window |
| **Background** — swatches + picker | Sets the overlay background. The fourth swatch is fully transparent — only the timer text floats |
| **Font Size** — Small / Medium / Large / XL | Changes the text size and automatically resizes the overlay window |
| **Opacity** — slider | Controls how transparent the entire overlay panel is (30 %–100 %) |

### Show Overlay / Hide Overlay
Toggles the floating overlay window. When hidden, the overlay is off-screen but retains its position and settings. Keyboard shortcut: **Cmd+O**.

## Using SlideTimer with PowerPoint or Keynote

**Recommended workflow:**

1. Open your presentation in PowerPoint / Keynote
2. Launch **SlideTimer** (it appears alongside your other windows)
3. Set your desired duration (e.g. click the **10m** preset)
4. Click **Show Overlay** — the floating timer window appears
5. Drag the overlay to your **secondary screen** (the one the audience doesn't see), or position it in a corner of your presenter screen
6. Start your slideshow in fullscreen
7. Click **Start** in the SlideTimer control window (or press **Space**)
8. The overlay floats **above all windows**, including fullscreen presentations — it stays visible even when PowerPoint/Keynote is in front
9. Clicking the overlay does **not** steal focus from your presentation

**Tip:** If you only have one screen, position the overlay in a corner that won't obscure your slides. Use **Opacity** to make it semi-transparent.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space` | Start / Pause toggle |
| `Cmd + R` | Reset timer to set duration |
| `Escape` | Reset timer to set duration |
| `Cmd + O` | Show / Hide overlay |

> Shortcuts work when the SlideTimer control window is focused.

## Tips

- **Transparent background** — select the "no fill" swatch (bottom-right in Background row) for a clean floating-text look. Great for dark slide backgrounds.
- **XL font for big rooms** — at 96 pt the overlay is large enough to read from a distance without the projector. Use on your own screen to monitor time while facing the audience.
- **Stopwatch mode** — useful for timed Q&A sessions where you want to count up rather than down.
- **Pause and resume** — the timer picks up exactly where it left off, no drift.
- **Multiple presets** — clicking a preset while the timer is running is disabled; reset first, then choose a new preset.
- **Resizing the overlay** — drag the edges of the overlay window to resize it freely, or use the Font Size picker to snap to standard sizes.
- **Quitting** — closing the control window quits the app. The overlay closes automatically.
